## üöÄ Quick Start

### 1. Clone & Install
```bash
git clone https://github.com/Navneet-55/new1.git
cd new1
npm install
```

### 2. Configure
```bash
cp .env.example .env.local
# Add GROQ_API_KEY=gsk_your_key_here
```

### 3. Run
```bash
npm run dev
# Open http://localhost:3000
```

### 4. Deploy
```bash
npm run build
npm start
```
